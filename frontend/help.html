Quick Start

  1. Click New Vault to start a new world, or Load Vault to open an existing vault.
  2. Open World to start building topics and notes for your setting.
  3. Use the left sidebar to create and organize topics.
  4. Edit content in the center editor and use the right panel for metadata, links, and tools.
  5. Use the editor toolbar for spelling and grammar, AI rewrite, TTS playback, and WAV export.
Changes auto-save shortly after edits. Save As creates a copy at a new location.

Workspace Layout

Top Bar

  • Category tabs: World, Characters, Timeline, Maps.
  • Left and right panel toggles: « and ».
  • Vault controls: New Vault, Load Vault, Save As.
  • AI controls: Use Local AI, AI Models, Scan File, Scan Folder.
  • Info and style controls: About, Settings.
  • Vault name display: shows the currently loaded file, or No vault loaded when working in a fresh unsaved vault.

Panels

  • Left panel: item tree for the current category.
  • Center panel: rich editor, timeline view, map view, character tabs.
  • Right panel: metadata, linked items, backlinks, marker editor.
You can drag the vertical dividers to resize panels. The top-bar toggle buttons collapse and restore the left sidebar or right metadata panel.
The editor header shows the current project and world titles. Click them to rename when available.

Vaults and Saving

  • New Vault: creates a fresh vault in memory.
  • Load Vault: opens a vault file from disk.
  • Save As: writes a copy and updates the active vault path.
  • Last vault path is remembered and auto-loaded on next launch when available.
A safety backup file <vault>.bak is created next to your vault periodically during saves.
The desktop menu also includes Save Vault, Find & Replace, Rebuild Vault Index, and export actions for the current tab or the full vault.
Most tools follow the current category, so the sidebar and right panel update depending on whether you are in World, Characters, Timeline, or Maps.

World (Topics)

Create and Organize

  1. Open World.
  2. In the left sidebar, use the World section and click + Topic for a root topic.
  3. Right-click a topic for Add Child Topic, Rename, Delete.
  4. Drag topics to reorder, nest, or move between levels.

Topic Metadata

  • Topic name field (live updates tree and title).
  • Per-topic editor background color with Use Theme Default.
  • Linked items: characters, topics, maps, markers, timeline events.
  • Backlinks: Referenced In list with click-to-navigate.

Characters

Character Metadata

  • Name fields, race, age, gender, class, personality, background, motivations.
  • Portrait options: emoji and uploaded image, plus clear image.
  • Tags (comma-separated input).
  • Linked items: characters, topics, maps, markers, events.
  • Backlinks section (Referenced In).

Character Center Tabs

  • Overview: summary view.
  • Biography: full rich editor.
  • Notes: separate notes editor.
  • Relationships: add/edit target character, relationship type, notes.
Most character fields auto-save as you type.

Timeline Events

Event Workflow

  1. Open Timeline and click + Add.
  2. Edit event description in the center rich editor.
  3. Set Date, End Date, and Category in the right panel.
  4. Use linked-item dropdowns for characters, topics, maps, markers.

Timeline View

  • Shows event cards with date range and category.
  • Click a card to open that event.
  • Right panel also shows backlinks and a simplified relationship network.

Maps and Markers

Create and Configure a Map

  1. Open Maps and add a map item.
  2. Use Replace Image or Change Map Image in metadata.
  3. Map metadata panel shows current image path and unplaced markers.

Map Modes and Marker Editing

  • Add Marker cycles map mode between pan/add/measure states.
  • In add mode, click the map to place a marker.
  • Drag placed markers to move them.
  • Right-click marker menu: edit, rename, recolor, add label, copy coordinates, delete.
  • Marker editor fields: name, description, color, category/label.
  • Marker entity linking: character/topic/event with status and unlink button.

Unplaced Markers

  • Unplaced markers appear in the right panel list.
  • Use Place on Map to place next click, or delete from the list.

Distance and Travel Time

Basic Measuring

  1. Click 📏 Measure Distance in map toolbar.
  2. Click at least two markers to build a route.
  3. Read results in the map toolbar: walking and horse travel time.
  4. Use Clear Path to reset.

Measure Options

  • Distance method: follow route segments or direct start-to-end line.
  • On-foot pace in map units/day.
  • Horse presets: Standard, Careful, Forced, or Custom multiplier.
  • Auto-calculate mounted pace from walking pace (or manual horse pace).
  • Time output format: weeks, days, hours, minutes.

Calibrate From a Known Trip

  1. Measure a route first (2+ points).
  2. Open ⚙ Measure Options.
  3. In Known trip duration, enter how long that measured route should take, e.g. 1 week or 4 days 6 hours.
  4. Click Set On-foot Pace From Route.
  5. Click Apply to save settings.

Rich Text Editor

Formatting Toolbar

  • Undo/redo.
  • Font family + font size (preset sizes plus a custom pixel input with OK / Cancel controls).
  • Bold, italic, underline, strikethrough.
  • Text color and background color, plus clear background.
  • Headings H1/H2/H3.
  • Align left/center/right.
  • Bullet list, numbered list, quote.
  • Spelling & grammar (offline): ABC✓ opens LanguageTool suggestions.
  • AI rewrite (offline, local model): AI✎ rewrites selected text or the current paragraph while preserving voice.
  • Text to speech: TTS reads the current selection, or the active editor if nothing is selected.
  • Speech export: WAV saves audio from bundled Piper as a WAV file.
LanguageTool runs locally (127.0.0.1) and requires Java. The in-app Download/Update LT button only downloads when you click it.

Images

  • Insert image from URL or file picker.
  • Alignment buttons: left, center, right, inline.
  • Images are resizable by dragging handles/edges.

Hyperlinks

  • Use 🔗 to insert or edit a link in selected text.
  • Link types include external URLs and internal world-item links.
  • The hyperlink modal supports link text, optional title tooltip, open-in-new-tab, and nofollow for external links.
  • You can remove an existing link from the same modal.

Tables

  • Insert table modal: rows, columns, border width/color, cell background.
  • Table controls appear when cursor is inside a table.
  • Operations include row/column add/delete, merge/split, header toggle, border/cell styling.
  • Table border preview toggle (👁️▦) hides borders when not editing.
Exports to DOCX include tables and embedded images.

Shapes Tool

  • Insert rectangle/circle/triangle with fill + border options.
  • Can insert shape as image or as table-cell background.
  • Background fit modes: stretch, custom size, contain, cover, tile.

Review, Rewrite, and Text To Speech

LanguageTool (Offline)

  • Click ABC✓ to open offline spelling and grammar tools.
  • Set the language, embedded JAR path, and local server port from the modal.
  • Use Check Current Editor or Check Selection.
  • Use Download/Update LT to install or refresh the local LanguageTool bundle.
  • Custom dictionary entries are tracked separately and can be cleared from the modal.
  • Suggestions can be applied one-by-one directly from the results list.

AI Rewrite (Local Model)

  • Click AI✎ to rewrite the current selection or paragraph using your local GGUF model.
  • Modes include Fix grammar (minimal) and Rewrite (preserve voice).
  • Use the strength slider to control how aggressive the rewrite should be.
  • Review the suggestion before applying it back into the editor.

Text To Speech

  • Open Settings to choose the TTS provider: bundled Piper or system voices.
  • Bundled Piper models are discovered from the app bundle and listed automatically.
  • Delivery presets include Raw, Natural, and Narration.
  • Grammar cleanup and pause strength can improve read-aloud pacing before playback.
  • Use the speed and pitch sliders plus Preview Voice to test your setup.
  • Piper WAV export uses bundled voices only. System voices are playback-only.
For proof-listening, run LanguageTool first if your punctuation is rough, then use TTS with Natural or Narration.

Linking System

Text Links in Articles

  1. Select text in the editor (example: Queen Anna).
  2. Click 🔗 Insert Link.
  3. Choose link type:
    • Website URL, Email, Phone, Page Anchor
    • World Item (Internal) for Topics/Characters/Events/Markers
  4. Choose display mode: text link or button style.
  5. Click Apply (or Remove Link).

Previews and Navigation

  • Hover linked items in metadata lists to preview basic information before navigating.
  • Relationship views in event metadata provide a simplified network of connected items for quick navigation.

Linked Items + Backlinks

  • Right panel link pills navigate to linked items.
  • × removes a link from the current item.
  • Backlink sections show where the current item is referenced.
  • Internal links inside editor content are clickable and navigate directly.

Scanner and Local AI

Scan File / Scan Folder

  1. Use Scan File or Scan Folder in top bar.
  2. Text must be at least 50 characters.
  3. Review suggestions modal for characters, events, topics, and existing-entity matches.
  4. Use per-item Add Now or Apply All Checked.
  5. Use Rescan to rerun with current settings.

Use Local AI (Optional)

  • Enable Use Local AI toggle before scanning.
  • Open AI Models to manage local models.
  • Model manager supports:
    • Install llama-cpp-python
    • Download/delete supported models
    • Select active installed model
  • Local AI runs on your machine using GGUF models; no text is sent online.

Settings, Themes, Templates, and TTS

  1. Open 🎨 Settings.
  2. Select a built-in color scheme.
  3. Adjust individual color tokens in Custom Colors.
  4. Use Import Theme, Export Theme, or Reset.
  • Editor font family and size are configurable from the editor toolbar.
  • Theme and font choices are persisted.
  • Settings also include default templates for new topics and new characters.
  • You can apply the selected template to the current item, save the current topic/character as a custom template, or delete a selected custom template.
  • Template selectors also show the active default template and its description before you apply it.
  • TTS settings include provider, system voice, Piper model, delivery preset, grammar cleanup, pause strength, speed, pitch, and voice preview.
  • Topic and character templates can be set as defaults, applied to the current item, or captured from the current item.
Theme settings are local UI preferences. Topic and character template choices persist across sessions.

Desktop Menu

Keyboard and Interaction Shortcuts

  • F1: open this help page.
  • F12: open the internal debug console in the Linux packaged build.
  • Esc: close modals; exits focus mode; exits fullscreen if active.
  • In modals: Enter applies/accepts; Esc closes.
  • Table shortcuts (cursor in a table cell):
    • Ctrl/Cmd + Alt + ArrowUp: add row above
    • Ctrl/Cmd + Alt + ArrowDown: add row below
    • Ctrl/Cmd + Alt + ArrowLeft: add column left
    • Ctrl/Cmd + Alt + ArrowRight: add column right
    • Ctrl/Cmd + Alt + M: merge cell
    • Ctrl/Cmd + Alt + S: split cell
    • Ctrl/Cmd + Alt + H: toggle header row
    • Ctrl/Cmd + Shift + A: select all cells in current table
There is no dedicated Save button in the top bar. Saving is automatic, and Save As is available from the top bar.
