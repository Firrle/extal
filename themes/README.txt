// README for Example Theme Files
// 
// These are pre-built theme files that can be imported directly into the application
// 
// HOW TO USE:
// 1. Open the Settings modal (🎨 Settings button)
// 2. Click "📥 Import Theme"
// 3. Select any of these .json files
// 4. The theme will apply immediately
//
// FILES INCLUDED:
// - dark-default.json : Warm, muted dark theme (app default)
// - light.json : Clean, bright light theme
// - ocean.json : Cool blue ocean-inspired theme
// - cyberpunk.json : Bold neon cyberpunk theme
// - ember-triad.json : Three-color black/gray/orange theme
//
// CREATING YOUR OWN:
// 1. Customize colors in the Settings modal
// 2. Click "📤 Export Theme"
// 3. Your custom theme will download as a JSON file
// 4. Share, backup, or use on other devices
//
// FILE FORMAT:
// {
//   "scheme": "schemeName",
//   "custom": {
//     "bg-primary": "#hexcolor",
//     "text-primary": "#hexcolor",
//     ... (all 13 colors)
//   }
// }
